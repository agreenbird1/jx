#### 觉晓法考

参考觉晓法考官网：https://www.juexiaotime.com/ - 专注于在职法考教育。

本项目是觉晓法考的阉割版本，实现了登录（微信登录、手机号登录）；分章节做题；搜索关键字做题；做题批改解析等功能。

如何运行：

```bash
git clone http://124.71.154.156:9090/afanti/afanti-project.git
cd afanti-project
# node -v v16.15.1
# pnpm -v 6.32.24
pnpm install
# 切换到对应的分支
pnpm run dev
```
